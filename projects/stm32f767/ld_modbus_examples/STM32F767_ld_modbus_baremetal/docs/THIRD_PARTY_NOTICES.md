# 第三方组件说明

本工程包含以下第三方或外部来源代码：

- `Middlewares/ld_modbus`：`l993889589/ld_modbus`，Apache License 2.0；
- `Drivers/STM32F7xx_HAL_Driver`：STMicroelectronics STM32 HAL，许可见各源码文件头；
- `Drivers/CMSIS`：Arm CMSIS 与 ST 设备支持文件，许可见各源码文件头；

本仓库对外部代码的来源只作事实说明，不改变其原有许可证。工程构建产物、原附件 ZIP 和本地 IDE 状态不随源码发布。
